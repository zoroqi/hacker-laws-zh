# 诉诸后果

[诉诸后果](https://zh.wikipedia.org/wiki/%E8%A8%B4%E8%AB%B8%E5%BE%8C%E6%9E%9C) 是一种非形式谬误，主张由于某结果是好的，因此它是对的；或某结果是坏的，因此是这件事是错的。

- 正面形式: 如果P成立，会导致Q; Q是好的; 因此，P成立. 与 [[wishful-thinking|一厢情愿]] 相似.
- 负面形式: 如果P成立，会导致Q; Q是不好的; 因此，P不成立.
