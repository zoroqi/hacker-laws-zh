# 雷斯多夫效应

雷斯多夫效应（Von Restorff effect，又称“莱斯托夫效应”）是指个人对学习材料或所见所闻的资讯，容易记住最特殊的部分的现象。例如：有一些参考书将重要的数据，以不同颜色或特殊的字体标示出来，就是利用雷斯多夫效应来加深读者的印象。

醒目的事或特别强调的事比其他的事更容易记住。
