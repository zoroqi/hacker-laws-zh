# 量化词逻辑谬误

谓词逻辑是对命题逻辑的扩充，主要引入了量化词运算子“对于所有的”（ ∀ {\displaystyle \forall } ）、“存在”（ ∃ {\displaystyle \exists } ）。误用量化词相关运算规则即构成量化词逻辑谬误。
