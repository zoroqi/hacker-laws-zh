# 基本比率谬误

只关注针对性的资讯，忽略一般性的资讯（基本比率），导致不恰当的认知。例如因为酒驾较容易肇事，就认为肇事者多半是酒驾，然而由于酒驾者占所有驾驶者的比率甚小，肇事者之中酒驾者的比例不会那么高。
