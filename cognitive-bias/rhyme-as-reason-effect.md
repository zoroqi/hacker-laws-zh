# 韵律当理由效应

Rhyme as Reason Effect，人们会更容易把有韵律的、押韵的句子，当作是真实和准确的。一句话只要变得有韵律了，那么就更有说服力了。

应用：大部分广告语都朗朗上口，非常洗脑。所以，尽可能的把你的产品文案变得有韵律感，这样更容易说服用户。
