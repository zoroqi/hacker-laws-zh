# 集群错觉

过度期待从小样本或小型测试中发现的规律，然而这些小样本或小型测试是从大样本随机抽取，大样本往往并无这种规律（[[sampling-bias|取样偏差]]）。

**Clustering Illusion**: We find patterns and “clusters” in random data.
聚类错觉：我们会在原本随机的数据信息中发现模式和规律。
