# 转换编程法

* [wiki:https://en.wikipedia.org/wiki/Programming_by_permutation](https://en.wikipedia.org/wiki/Programming_by_permutation)

也称巧合编程，试图通过连续修改代码再看是否工作的方式来解决问题

> [!NOTE]
> 调试一个完全不知道如何运行的代码, 这就是一个必然的方法.
