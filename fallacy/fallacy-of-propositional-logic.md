# 命题逻辑谬误

命题逻辑是把命题视做不可分割的整体，对多个命题做真假值运算的形式系统，主要运算子包括（非； ¬ {\displaystyle \lnot } ）、联言（且； ∧ {\displaystyle \land } ）、选言（或； ∨ {\displaystyle \lor } ）、蕴涵（若； → {\displaystyle \rightarrow } ）、等值（当且仅当； ↔ {\displaystyle \leftrightarrow } ）。违反命题逻辑系统运算规则即构成命题逻辑谬误。
